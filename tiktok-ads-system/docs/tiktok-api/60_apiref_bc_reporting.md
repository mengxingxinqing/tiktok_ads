# API Reference - BC Reporting

Business Center is a one-stop business hub for organizations of all sizes to activate marketing activities and reach desired audiences in a collaborative way on TikTok. BC Reporting API enables you to retrieve the supported values of the currency and place of registration filters that can be used in a Business Center report for a specific Business Center.
