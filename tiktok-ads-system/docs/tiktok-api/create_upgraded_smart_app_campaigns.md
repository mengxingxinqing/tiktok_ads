# Create Upgraded Smart+ App Campaigns

**Doc ID**: 1843309500205058
**Path**: Use Cases/Campaign creation/Create an Upgraded Smart+ Campaign/Create Upgraded Smart+ App Campaigns

---

This article walks you through the steps to create Upgraded Smart+ App Campaigns using the App Promotion objective.

Based on your preference for creating an iOS 14 Dedicated Campaign (DC) and whether to promote a TikTok Minis, you can create an Upgraded Smart+ non-DC App Campaign, an Upgraded Smart+ DC App Campaign, or an Upgraded Smart+ Minis Campaign. 

- To create an Upgraded Smart+ non-DC App Campaign without promoting a TikTok Minis, see [Create an Upgraded Smart+ non-DC App Campaign](https://business-api.tiktok.com/portal/docs?id=1853377390815233).
- To create an Upgraded Smart+ DC App Campaign without promoting a TikTok Minis,  see [Create an Upgraded Smart+ DC App Campaign](https://business-api.tiktok.com/portal/docs?id=1853377803841537).
- To create an Upgraded Smart+ non-DC App Campaign promoting a TikTok Minis, see [Create an Upgraded Smart+ Minis Campaign](https://business-api.tiktok.com/portal/docs?id=1853377811982657).
