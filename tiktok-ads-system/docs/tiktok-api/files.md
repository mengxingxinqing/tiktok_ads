# Files

**Doc ID**: 1737719988918274
**Path**: API Reference/Files

---

You can upload video, image and music files to TikTok's temporary file repository. Files in the repository can be transferred to TikTok’s Asset Library and then used in your ads. Once a file is uploaded to the file repository, it is kept for 24 hours, after which it will expire. For a file that is larger than 20 MB, you can segment it into chunks and then upload the chunks to the file repository.

See [Upload videos, images and music-Upload video, image and music files](https://ads.tiktok.com/marketing_api/docs?id=1738963828459521) to learn about the steps for uploading a single file or file chunks to the file repository.

This group consists of the following endpoints:
* [Upload a file](https://ads.tiktok.com/marketing_api/docs?id=1737720893061122)
* [Start a chunk upload task](https://ads.tiktok.com/marketing_api/docs?id=1737721229896705)
* [Transfer a file chunk](https://ads.tiktok.com/marketing_api/docs?id=1737721489945601)
* [Finish a chunk upload task](https://ads.tiktok.com/marketing_api/docs?id=1737721642924033)
* [Check the names of files](https://ads.tiktok.com/marketing_api/docs?id=1759130033155073)
