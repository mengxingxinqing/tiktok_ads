# GMV Max

**Doc ID**: 1822000911166465
**Path**: API Reference/GMV Max

GMV Max is an automated ad campaign type that optimizes total channel ROI for your TikTok Shop. 

To learn about how to create such campaigns, see [Create GMV Max Campaigns](https://business-api.tiktok.com/portal/docs?id=1822009058467842).
