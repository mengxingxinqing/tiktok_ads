# Smart Creative

**Doc ID**: 1739472835319809
**Path**: API Reference/Smart Creative

---

Smart Creative is a TikTok automated creative solution that optimizes ad creative combinations through ad group fatigue detection and auto-refresh strategies. This feature reduces ad fatigue, prolongs ad group lifespan, and improves cost efficiency over time.

To learn about how to enable Smart Creative for your ad groups, see [Smart Creative](https://business-api.tiktok.com/portal/docs?id=1767322784934914).

This group consists of the following endpoints: 
* [Create Smart Creative ads](https://ads.tiktok.com/marketing_api/docs?id=1739473063234626)
* [Get Smart Creative materials](https://ads.tiktok.com/marketing_api/docs?id=1739473020978177)
* [Update Smart Creative materials](https://ads.tiktok.com/marketing_api/docs?id=1739473077112833)
* [Update the statuses of Smart Creative materials](https://ads.tiktok.com/marketing_api/docs?id=1739506701165570)
