# Ad groups

**Doc ID**: 1843310524500993
**Path**: API Reference/Upgraded Smart+/Ad groups

---

Ad groups are the middle level of advertising. You can set specific ad placements, target audiences, budgets, schedule, bids, and optimization goals for each ad group at the ad group level.

This group consists of the following endpoints:

* [Get Upgraded Smart+ Ad Groups](https://business-api.tiktok.com/portal/docs?id=1843314879617026)
* [Create an Upgraded Smart+ Ad Group](https://business-api.tiktok.com/portal/docs?id=1843314887930946)
* [Update an Upgraded Smart+ Ad Group](https://business-api.tiktok.com/portal/docs?id=1843314894279682)
* [Update the operation statuses of Upgraded Smart+ Ad Groups](https://business-api.tiktok.com/portal/docs?id=1843314908153858)
* [Update the budgets of Upgraded Smart+ Ad Groups](https://business-api.tiktok.com/portal/docs?id=1843314914438466)
