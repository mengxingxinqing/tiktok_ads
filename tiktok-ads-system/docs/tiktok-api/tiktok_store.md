# TikTok Store

**Doc ID**: 1740661959804929
**Path**: API Reference/TikTok Store

---

A TikTok store is an entity that merchants can use to manage and sell products on TikTok. TikTok Store API enables you to retrieve first-party stores (TikTok Shops) under an ad account and products within TikTok Shops.

See [Get started](https://ads.tiktok.com/marketing_api/docs?id=1739581925704769) to learn about the roles and concepts commonly used in TikTok Store and the general workflow.

This group consists of the following endpoint:
* [Get available stores under an ad account](https://ads.tiktok.com/marketing_api/docs?id=1752267762718722)
* [Get products within a TikTok Shop](https://business-api.tiktok.com/portal/docs?id=1793482248880130)
