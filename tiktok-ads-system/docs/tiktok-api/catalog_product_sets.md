# Catalog Product Sets

**Doc ID**: 1740132953884673

Please review these policies and ensure that the items you upload to your catalog don't breach them.

A catalog is an asset that allows you to store information about the products you want to promote on TikTok. Using a catalog is a great solution for anyone who is looking for an efficient way to create and manage ads that feature a variety of products, like an online retailer. Catalog Product Sets API enables you to manage product sets in a catalog under your Business Center.

See [Overview](https://ads.tiktok.com/marketing_api/docs?id=1754263078057986) and [Guides](https://ads.tiktok.com/marketing_api/docs?id=1754263100123137) to learn about the concepts and benefits of catalogs, and to find out how to create and manage catalogs.

This group consists of the following endpoints:
* [Get product sets](https://ads.tiktok.com/marketing_api/docs?id=1740570556295169)
* [Get products in product set](https://ads.tiktok.com/marketing_api/docs?id=1740571478441986)
* [Create a product set by conditions](https://ads.tiktok.com/marketing_api/docs?id=1740572891104257)
* [Create a product set by file](https://ads.tiktok.com/marketing_api/docs?id=1846770644217858)
* [Update a product set](https://ads.tiktok.com/marketing_api/docs?id=1740572974725122)
* [Delete product sets](https://ads.tiktok.com/marketing_api/docs?id=1740573143966722)
