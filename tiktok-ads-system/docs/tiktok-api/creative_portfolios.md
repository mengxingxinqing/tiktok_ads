# Creative Portfolios

**Doc ID**: 1739091934311426
**Path**: API Reference/Creative Portfolios

---

A creative portfolio is a combination of creative assets like CTAs and Download Cards. You can include creative assets in a creative portfolio, and then use the portfolio in an ad.

This group consists of the following endpoints:
* [Create a portfolio](https://ads.tiktok.com/marketing_api/docs?id=1739091950439426)
* [Get a portfolio by ID](https://ads.tiktok.com/marketing_api/docs?id=1739092113671170)
* [Get portfolios within an ad account](https://ads.tiktok.com/marketing_api/docs?id=1766324010279938)
* [Delete portfolios](https://business-api.tiktok.com/portal/docs?id=1834560049625089)
