# Campaigns

**Doc ID**: 1843310518576130
**Path**: API Reference/Upgraded Smart+/Campaigns

---

Campaigns are the highest level of advertising. To promote on TikTok for Business, you first need to create a campaign and set your advertising objective.

This group consists of the following endpoints:

* [Get Upgraded Smart+ Campaigns](https://business-api.tiktok.com/portal/docs?id=1843312818332930)
* [Create an Upgraded Smart+ Campaign](https://business-api.tiktok.com/portal/docs?id=1843312852800706)
* [Update an Upgraded Smart+ Campaign](https://business-api.tiktok.com/portal/docs?id=1843312876001282)
* [Update the operation statuses of Upgraded Smart+ Campaigns](https://business-api.tiktok.com/portal/docs?id=1843312888885314)
