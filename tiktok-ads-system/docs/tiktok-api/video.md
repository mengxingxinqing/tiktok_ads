# Video

**Doc ID**: 1737587293087810
**Path**: API Reference/Video

---

Videos are a type of creative assets you can use to create an ad. You can also use the video ID in creative tools such as [Smart Video](https://ads.tiktok.com/marketing_api/docs?id=1739352294717441),  [Smart Video Soundtrack](https://ads.tiktok.com/marketing_api/docs?id=1739089635852289), [Quick Optimization](https://ads.tiktok.com/marketing_api/docs?id=1739355078757377), and [Smart Fix](https://ads.tiktok.com/marketing_api/docs?id=1741468875279361). In addition, videos can be included in a video package via [/catalog/video_package/create/](https://ads.tiktok.com/marketing_api/docs?id=1740665273039873), where you would need to specify the video via `video_id`.

This group consists of the following endpoints:
* [Upload a video](https://ads.tiktok.com/marketing_api/docs?id=1737587322856449)
* [Update the name of a video](https://ads.tiktok.com/marketing_api/docs?id=1740049987749890)
* [Get info about videos](https://ads.tiktok.com/marketing_api/docs?id=1740050161973250)
* [Search for videos](https://ads.tiktok.com/marketing_api/docs?id=1740050472224769)
* [Get suggested thumbnails for a video](https://ads.tiktok.com/marketing_api/docs?id=1740051189071873)
