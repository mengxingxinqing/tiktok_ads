# Create Upgraded Smart+ Web Campaigns

**Doc ID**: 1843309514567809
**Path**: Use Cases/Campaign creation/Create an Upgraded Smart+ Campaign/Create Upgraded Smart+ Web Campaigns

---

This article walks you through the steps to create Upgraded Smart+ Web Campaigns.

Based on your preference for the specific type of web campaign and the catalog type, you can create regular Upgraded Smart+ Web Campaigns, Upgraded Smart+ E-commerce Catalog Ads, Upgraded Smart+ Travel Ads, Upgraded Smart+ Streaming Ads, or Upgraded Smart+ Mini Series Catalog Ads. 

- To create an Upgraded Smart+ Web Campaign without using a catalog, see [Create a regular Upgraded Smart+ Web Campaign](https://business-api.tiktok.com/portal/docs?id=1847302878676994).
- To create an Upgraded Smart+ Web Campaign using an E-commerce catalog,  see [Create Upgraded Smart+ E-commerce Catalog Ads](https://business-api.tiktok.com/portal/docs?id=1847302895272962).
- To create an Upgraded Smart+ Web Campaign using a travel (hotel, flight, or destination) catalog, see [Create Upgraded Smart+ Travel Ads](https://business-api.tiktok.com/portal/docs?id=1847302908327938).
- To create an Upgraded Smart+ Web Campaign using an entertainment catalog, see [Create Upgraded Smart+ Streaming Ads](https://business-api.tiktok.com/portal/docs?id=1847302920924306).
- To create an Upgraded Smart+ Web Campaign using a mini series catalog, see [Create Upgraded Smart+ Mini Series Catalog Ads](https://business-api.tiktok.com/portal/docs?id=1847302935967746).
