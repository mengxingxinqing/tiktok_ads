A catalog is an asset that allows you to store information about the products you want to promote on TikTok. Using a catalog is a great solution for anyone who is looking for an efficient way to create and manage ads that feature a variety of products, like an online retailer. Catalog Insights API enables you to obtain insights about trending products in your catalog and trending catalog product categories on TikTok.

See [Overview](https://ads.tiktok.com/marketing_api/docs?id=1754263078057986) and [Guides](https://ads.tiktok.com/marketing_api/docs?id=1754263100123137) to learn about the concepts and benefits of catalogs, and to find out how to create and manage catalogs. To learn how to access Catalog Product Insights, see [Obtain Catalog Product Insights](https://business-api.tiktok.com/portal/docs?id=1805642743426050).

This group consists of the following endpoints:
- [Get filters for catalog product insights](https://business-api.tiktok.com/portal/docs?id=1805640864553985)
- [Get trending catalog products](https://business-api.tiktok.com/portal/docs?id=1805640886872066)
- [Get trending catalog product categories](https://business-api.tiktok.com/portal/docs?id=1805640900563969)
