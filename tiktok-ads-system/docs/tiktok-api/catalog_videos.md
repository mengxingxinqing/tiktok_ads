A catalog is an asset that allows you to store information about the products you want to promote on TikTok. Using a catalog is a great solution for anyone who is looking for an efficient way to create and manage ads that feature a variety of products, like an online retailer. Catalog Videos API enables you to upload, retrieve and delete catalog videos within your catalog.

To learn more about how to manage uploaded catalog videos to be used in VSA for Catalog, see [Upload catalog videos to associate with catalog products](https://business-api.tiktok.com/portal/docs?id=1803654904260674).

This group consists of the following endpoints:
* [Upload catalog videos via a file URL](https://business-api.tiktok.com/portal/docs?id=1803655037415489)
* [Get the catalog video handling log](https://business-api.tiktok.com/portal/docs?id=1803655061642242)
* [Get the uploaded catalog videos within a catalog](https://business-api.tiktok.com/portal/docs?id=1803655082498050)
* [Delete uploaded catalog videos](https://business-api.tiktok.com/portal/docs?id=1803655103069185)
