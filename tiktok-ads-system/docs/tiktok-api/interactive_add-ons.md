# Interactive Add-ons

**Doc ID**: 1749019605861377
**Path**: Marketing API/Creatives/Guides/Creative assets/Interactive Add-ons

---

Interactive Add-ons capture the undivided attention of your audience in a full-screen, sound-on environment to boost brand engagement and encourage interaction. 

We support the following types of Interactive Add-Ons: [Cards](https://ads.tiktok.com/marketing_api/docs?id=1749019652141058), [Stickers](https://ads.tiktok.com/marketing_api/docs?id=1749019667506177) and [Premium Add-ons](https://ads.tiktok.com/marketing_api/docs?id=1749019676181505).
