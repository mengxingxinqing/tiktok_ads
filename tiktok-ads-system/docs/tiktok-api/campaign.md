# Campaign

**Doc ID**: 1739315728330753
**Path**: API Reference/Campaign

---

Campaigns are the highest level of advertising. To promote on TikTok for Business, you first need to create a campaign and set your advertising objective.

See [Guides-Campaign](https://ads.tiktok.com/marketing_api/docs?id=1745292231199745) to find out the procedure for creating a campaign and learn about the concepts of advertising objectives, budget, Reach & Frequency, and iOS 14 Dedicated Campaign. See [Ad structure](https://ads.tiktok.com/marketing_api/docs?id=1739381193120770) to find out the three advertising levels.

This group consists of the following endpoints: 

* [Get campaigns](https://ads.tiktok.com/marketing_api/docs?id=1739315828649986)
* [Create a campaign](https://ads.tiktok.com/marketing_api/docs?id=1739318962329602)
* [Update a campaign](https://ads.tiktok.com/marketing_api/docs?id=1739320422086657)
* [Update the statuses of campaigns](https://ads.tiktok.com/marketing_api/docs?id=1739320994354178)
* [Get the quota for a Dedicated Campaign per ad network](https://ads.tiktok.com/marketing_api/docs?id=1752256376677378)
* [Create an asynchronous campaign copy task](https://business-api.tiktok.com/portal/docs?id=1788434394151938)
* [Get the results of an asynchronous campaign copy task](https://business-api.tiktok.com/portal/docs?id=1788434463507458)
