# API Reference - Files

---

You can upload video, image and music files to TikTok's temporary file repository. Files in the repository can be transferred to TikTok’s Asset Library and then used in your ads. Once a file is uploaded to the file repository, it is kept for 24 hours, after which it will expire. For a file that is larger than 20 MB, you can segment it into chunks and then upload the chunks to the file repository.
