# Images

**Doc ID**: 1739067422728193
**Path**: API Reference/Images

---

Images can be divided into three categories: images to be used as ads, images as video covers, and profile images. See [Upload videos, images and music-Upload images](https://ads.tiktok.com/marketing_api/docs?id=1738963828459521) to learn about how to upload images for these three purposes.

Images are also a type of creative assets you can include in a creative portfolio via [/creative/portfolio/create/](https://ads.tiktok.com/marketing_api/docs?id=1739091950439426), where you would need to specify the image via `image_id`.

This group consists of the following endpoints:
* [Upload an image](https://ads.tiktok.com/marketing_api/docs?id=1739067433456642)
* [Update the name of an image](https://ads.tiktok.com/marketing_api/docs?id=1740039806891010)
* [Get info about images](https://ads.tiktok.com/marketing_api/docs?id=1740051721711618)
* [Search for images](https://ads.tiktok.com/marketing_api/docs?id=1740052016789506)
