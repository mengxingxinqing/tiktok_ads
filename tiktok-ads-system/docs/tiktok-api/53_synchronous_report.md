The table below presents an overview of these report types, along with their compatibility with auction ads and reservation ads. Report type Description Support auction ads Support reservation ads Basic report Use this type of report to get spending and performance data at four levels: campaigns, ad groups, ads, or ad accounts. For details, see Note : For basic reports, you need to set service_type to AUCTION and report_type BASIC . In contrast, for reservation ad reports, you need to set RESERVATION : You can retrieve data of TopView ads from basic reports. Audience report Use this type of report to get audience data. You can group spend and performance data by audience attributes such as age, gender, region, or interest. : You can retrieve data of TopView ads from audience reports. Playable ad report Use this type of report to get data about playable ads. DSA report Use this type of report to get data about Dynamic Showcase Ads (Dynamic Product Ads) data. Business Center report Use this type of report to get spending and performance data of ad accounts in a Business Center. GMV max ads report Use this type of report to get spending data on Product GMV max ads and LIVE GMV max ads created from TikTok Shop at two levels: campaign level and ad account level.
