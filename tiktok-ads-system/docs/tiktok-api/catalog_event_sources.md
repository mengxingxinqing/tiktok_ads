A catalog is an asset that allows you to store information about the products you want to promote on TikTok. Using a catalog is a great solution for anyone who is looking for an efficient way to create and manage ads that feature a variety of products, like an online retailer. An event source is an app event or a website event that you bind to a catalog in a Business Center. Catalog Event Sources API enables you to bind an event source to a catalog, unbind an event source from a catalog, and get the event source binding information of a catalog.

See [Overview](https://ads.tiktok.com/marketing_api/docs?id=1754263078057986) and [Guides](https://ads.tiktok.com/marketing_api/docs?id=1754263100123137) to learn about the concepts and benefits of catalogs, and to find out how to create and manage catalogs.

This group consists of the following endpoints: 

* [Bind an event source to a catalog](https://ads.tiktok.com/marketing_api/docs?id=1740492491200513)
* [Unbind an event source from a catalog](https://ads.tiktok.com/marketing_api/docs?id=1740492512449538)
* [Get event source binding info of a catalog](https://ads.tiktok.com/marketing_api/docs?id=1740492531343362)
