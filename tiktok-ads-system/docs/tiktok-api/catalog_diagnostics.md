A catalog is an asset that allows you to store information about the products you want to promote on TikTok. Using a catalog is a great solution for anyone who is looking for an efficient way to create and manage ads that feature a variety of products, like an online retailer. Catalog Diagnostics API enables you to obtain catalog diagnostic information for products or event sources. 

See [Overview](https://ads.tiktok.com/marketing_api/docs?id=1754263078057986) and [Guides](https://ads.tiktok.com/marketing_api/docs?id=1754263100123137) to learn about the concepts and benefits of catalogs, and to find out how to create and manage catalogs. To find out the steps of using Catalog Diagnostics, see [Troubleshoot catalogs with diagnostics](https://business-api.tiktok.com/portal/docs?id=1780803490136066) and [Mapping between Catalog Diagnostics features on Catalog Manager and API configurations](https://business-api.tiktok.com/portal/docs?id=1780803525320706).

This group consists of the following endpoints:

* [Get synchronous catalog product diagnostic information](https://ads.tiktok.com/marketing_api/docs?id=1771117232728066)
* [Create an asynchronous download task for catalog product diagnostic information](https://ads.tiktok.com/marketing_api/docs?id=1771117279175682)
* [Download asynchronous catalog product diagnostic information](https://ads.tiktok.com/marketing_api/docs?id=1771117294731266)
* [Get catalog event source diagnostic information](https://business-api.tiktok.com/portal/docs?id=1780799405041665)
* [Get catalog event trends and match rate](https://business-api.tiktok.com/portal/docs?id=1780802101608450)
