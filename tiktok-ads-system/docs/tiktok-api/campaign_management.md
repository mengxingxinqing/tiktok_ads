# Campaign Management

**Doc ID**: 1735713781404673
**Path**: Marketing API/Campaign Management

---

You can use Campaign Management API to set up your campaigns, ad groups, and ads.
> **Important**
For details about ad creation through API, refer to [Use Cases](https://business-api.tiktok.com/portal/docs?id=1735713713137730).
- To learn about campaign structure and the general workflow, see [Overview](https://ads.tiktok.com/marketing_api/docs?id=1737585544915970).
- To learn about the key settings and features at the campaign, ad group, and ad level, see [Guides](https://ads.tiktok.com/marketing_api/docs?id=1739381695768577). 
**See** [here](https://business-api.tiktok.com/portal/docs?id=1740673047618561) to learn about mapping relationships between campaign features on TikTok Ads Manager and API configurations.
- [Campaign](https://ads.tiktok.com/marketing_api/docs?id=1745292231199745): Campaigns are the highest level of advertising. To promote on TikTok for Business, you first need to create a campaign and set your advertising objective.
