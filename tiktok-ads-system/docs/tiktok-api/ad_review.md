# Ad Review

**Doc ID**: 1843310536069122
**Path**: API Reference/Upgraded Smart+/Ad Review

---

Ad Review API enables you to get the review information for ad groups and ads, including the review status, rejection reasons, and suggestions, or to request for a re-evaluation after rejection.

This group consists of the following endpoints:
* [Get the review info of Upgraded Smart+ Ads](https://business-api.tiktok.com/portal/docs?id=1843317465695233)
* [Get the review info of Upgraded Smart+ Ad creatives](https://business-api.tiktok.com/portal/docs?id=1843317474334722)
* [Appeal rejection of an Upgraded Smart+ Ad](https://business-api.tiktok.com/portal/docs?id=1843317522553106)
