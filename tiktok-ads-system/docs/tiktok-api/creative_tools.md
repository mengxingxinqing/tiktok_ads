# Creative Tools

**Doc ID**: 1737583624272897
**Path**: API Reference/Creative Tools

---

Creative tools provide a variety of simple and common assets (including images, videos and ads) preview, generation and optimization capabilities. See [Creative tools](https://ads.tiktok.com/marketing_api/docs?id=1740286998081538) to find out the brief introductions of the capabilities we offer.

This group consists of the following endpoints:
* [Edit an image](https://ads.tiktok.com/marketing_api/docs?id=1739067107903489)
* [Preview an ad or a creative](https://ads.tiktok.com/marketing_api/docs?id=1739403070695426)
* [Create a Smart Video task](https://ads.tiktok.com/marketing_api/docs?id=1739352294717441)
* [Share creative assets](https://ads.tiktok.com/marketing_api/docs?id=1773192725768193&rid=nmsgaz98b9)
* [Delete creative assets](https://business-api.tiktok.com/portal/docs?id=1797202997456897)
* [Get Smart Text recommendations](https://ads.tiktok.com/marketing_api/docs?id=1739084248002626)
* [Get recommended CTAs](https://ads.tiktok.com/marketing_api/docs?id=1739362202742785)
* [Create a Smart Fix task](https://ads.tiktok.com/marketing_api/docs?id=1741468875279361)
* [Get the results of a Smart Fix task](https://ads.tiktok.com/marketing_api/docs?id=1741469487859714)
* [Get Creative Fatigue Detection results](https://ads.tiktok.com/marketing_api/docs?id=1767568466842626)
