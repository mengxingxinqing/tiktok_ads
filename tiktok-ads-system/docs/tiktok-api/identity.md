# Identity

**Doc ID**: 1740133137507393
**Path**: API Reference/Identity

---

The Identity feature lets you choose how you present your business in your ads. Identity API enables you to create or delete a Custom User identity, get identities under an ad account or posts under an identity, and get information about an identity or TikTok posts. 

See [Identities](https://ads.tiktok.com/marketing_api/docs?id=1738958351620097) to learn more about the types of authorized identities.

This group consists of the following endpoints:
* [Create an identity](https://ads.tiktok.com/marketing_api/docs?id=1740654203526146)
* [Delete an identity](https://ads.tiktok.com/marketing_api/docs?id=1759155939236865)
* [Get the identity list](https://ads.tiktok.com/marketing_api/docs?id=1740218420781057)
* [Get info about an identity](https://ads.tiktok.com/marketing_api/docs?id=1740218453385217)
* [Get posts under an identity](https://ads.tiktok.com/marketing_api/docs?id=1740218475032577)
* [Get live videos under an identity](https://business-api.tiktok.com/portal/docs?id=1785238052792321)
* [Get music authorization info of a video](https://ads.tiktok.com/marketing_api/docs?id=1740218495869954)
* [Get info about TikTok posts](https://ads.tiktok.com/marketing_api/docs?id=1740218522178562)
