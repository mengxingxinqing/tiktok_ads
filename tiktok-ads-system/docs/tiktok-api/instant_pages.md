# Instant pages

**Doc ID**: 1739953260466178
**Path**: Marketing API/Creatives/Guides/Creative assets/Instant pages

---

An instant page is a fast-loading lightweight web page that loads up to 11 times faster on TikTok than a standard web page.

Currently, you can use the [TikTok Instant Page (TIP) Editor SDK](https://business-api.tiktok.com/portal/docs?id=1740307202170881) to create the following types of Instant Pages:
- Customized Page (or Custom Page): A customized instant page to provide a better experience for your target audience.
- Instant Form: An instant page that enables advertisers to collect leads directly within the TikTok app, allowing customers to share their contact information, answer your questions, and learn about your business.
- App Profile Pages: An instant page that is designed to give users an enriched experience with relevant app information and drive more conversions for devices with iOS 14.5+ through more onsite signals.
- Pop-up Form: An instant page that is sent in direct messages to collect leads.
After you create an Instant Page, you can get the page ID of the Instant Page by using the [/page/get/](https://business-api.tiktok.com/portal/docs?id=1820826387779586) endpoint. The page ID can be used to create an ad.

To learn more about how to use the TIP Editor SDK, see [Create instant pages using TIP Editor SDK](https://business-api.tiktok.com/portal/docs?id=1740307202170881#item-link-Create%20instant%20pages%20using%20TIP%20Editor%20SDK).
