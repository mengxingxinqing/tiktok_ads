# Upgraded Smart+

**Doc ID**: 1843301794874370
**Path**: API Reference/Upgraded Smart+

---

The Upgraded Smart+ Campaigns feature a new, unified workflow, known as the Upgraded Smart+ experience. This streamlined process enables you to access both [manual](https://business-api.tiktok.com/portal/docs?id=1739315728330753) and [Smart+](https://business-api.tiktok.com/portal/docs?id=1765314260115458) features when creating your campaign. [Learn more about upgrades to Smart+](https://ads.tiktok.com/help/article/about-updates-to-smart-plus).

* [Get Upgraded Smart+ Campaigns](https://business-api.tiktok.com/portal/docs?id=1843312818332930)
