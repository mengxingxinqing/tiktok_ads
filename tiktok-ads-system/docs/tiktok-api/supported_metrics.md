# Supported metrics

**Doc ID**: 1803073913733121
**Path**: Marketing API/Reporting/Guides/Report types/GMV max ads reports/Supported metrics

---

You can use the following metrics in GMV max ads reports. 

## Attribute metrics
Attribute metrics provide basic attributes of your campaigns.

```xtable
| Field {25%}| Type{10%}| Description{20%}| Detail{45%}|
|---|---|---|---|
| campaign_name | string | Campaign name | Supported at Campaign level (`data_level` is `AUCTION_CAMPAIGN`). |
```

## Basic data metrics
Basic data metrics provide data about spend and performance of your ads.

```xtable
| Field {25%}| Type{10%}| Description{20%}| Detail{45%}|
|---|---|---|---|
| spend | string | Total Cost | The estimated total amount you've spent on your ads during its schedule. |
| billed_cost | string | Net cost | The amount spent, excluding ad credit/coupons used. |
```
