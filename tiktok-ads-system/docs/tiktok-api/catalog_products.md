# Catalog Products

**Doc ID**: 1740132940622850
**Path**: API Reference/Catalog Products

> **Important**
By using Catalog API endpoints, you agree to the [Product Catalog Terms](https://ads.tiktok.com/i18n/official/article?aid=6765817026859499525) and certify that you are abiding by the [TikTok Advertising Policies](https://ads.tiktok.com/help/article?aid=6685586866860720134). Please review these policies and ensure that the items you upload to your catalog don't breach them.

A catalog is an asset that allows you to store information about the products you want to promote on TikTok. Using a catalog is a great solution for anyone who is looking for an efficient way to create and manage ads that feature a variety of products, like an online retailer. Catalog Products API enables you to manage products in a catalog under your Business Center.

See [Overview](https://ads.tiktok.com/marketing_api/docs?id=1754263078057986) and [Guides](https://ads.tiktok.com/marketing_api/docs?id=1754263100123137) to learn about the concepts and benefits of catalogs, and to find out how to create and manage products in catalogs.

This group consists of the following endpoints:
* [Upload products via a file URL](https://ads.tiktok.com/marketing_api/docs?id=1740496787164161)
* [Upload products via a JSON schema](https://ads.tiktok.com/marketing_api/docs?id=1740497429681153)
* [Update products](https://ads.tiktok.com/marketing_api/docs?id=1740562287852546)
* [Remove products](https://ads.tiktok.com/marketing_api/docs?id=1740562489236481)
* [Get products](https://ads.tiktok.com/marketing_api/docs?id=1740564136678402)
* [Get the product handling log](https://ads.tiktok.com/marketing_api/docs?id=1740570027173889)

>  **Note**

> If you use the four endpoints below to operate on **the same catalog**, you can make up to 1000 queries per hour. For additional requests, you'll get an error. You need to wait until the next hour before you can make new API requests.
