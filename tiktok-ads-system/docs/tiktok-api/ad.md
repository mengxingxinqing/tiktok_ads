# Ad

**Doc ID**: 1745292235571201
**Path**: Marketing API/Campaign Management/Guides/Ad

Ad is the content that is presented to the target audience. This may include video or image material, ad text and call-to-action texts.

**See [here](https://business-api.tiktok.com/portal/docs?id=1740673047618561) for details about mapping relationships between campaign features on TikTok Ads Manager and API configurations.**

- [Identities](https://ads.tiktok.com/marketing_api/docs?id=1738958351620097): Introduces supported authorized identities and related endpoints.

- [AIGC self-disclosure toggle](https://business-api.tiktok.com/portal/docs?id=1799186631031809): Introduces the supported advertising objectives and ad formats for the AIGC (Artificial Intelligence Generated Content) self-disclosure toggle and how to use the toggle.
