# Creative Reports

**Doc ID**: 1749092271949826
**Path**: API Reference/Creative Reports

---

Creative asset level reporting is essential to understand how an advertiser's creative assets resonate with their target audience. This information can then be used to improve future creatives and refine audiences. 

This group consists of the following sets of endpoints: 
- [Creative basic reports](https://ads.tiktok.com/marketing_api/docs?id=1740662135093314): Help you get basic performance metrics at the creative asset level as well as creative asset attributes like height, and width for non-Spark Ads in one ad account. 
- [Video Insights reports](https://business-api.tiktok.com/portal/docs?id=1789700050715650): Display the performance data of multiple videos across one or more ad accounts, encompassing both Spark Ads and non-Spark Ads.
- [Creative insights](https://ads.tiktok.com/marketing_api/docs?id=1738824477276162): Use data insights to create the best performing TikTok ad.
