# Ads广告创建 UI 指南

## 访问方式

后端运行后，在前端页面访问：
```
http://localhost:5173/ads/create
```

或者在导航菜单中点击 **"🎬 创建Ads广告"** 菜单项（需要先在路由中添加）

---

## 四步广告创建向导

### 步骤1：选择广告账户

![Step 1](./docs/step1-select-advertiser.png)

**操作：**
1. 系统自动加载已授权的广告账户
2. 点击选择要使用的账户（账户卡片会高亮）
3. 确保账户状态显示 ✅ 授权有效
4. 点击"下一步 →"进入步骤2

**提示：**
- 如果没看到账户，说明还没有授权
- 点击"去绑定投放户"前往 `/advertisers` 页面绑定

---

### 步骤2：上传创意

![Step 2](./docs/step2-upload-creative.png)

**选择创意类型：**
- 📹 **视频创意**（推荐）
  - 格式：MP4, MOV, AVI, WMV
  - 最大：2GB
  - 时长：3秒 - 60分钟

- 🖼️ **图片创意**
  - 格式：JPG, PNG, GIF, WEBP
  - 最大：30MB
  - 分辨率：最小 600x600

**上传方式：**
1. **点击上传区域** — 选择文件
2. **拖拽上传** — 将文件拖到上传区域

**优化建议：**
- 推荐竖屏：1080x1920
- 或横屏：1920x1080
- 视频 15-30 秒效果最好
- Hook Rate（前2秒）非常重要

---

### 步骤3：广告配置

![Step 3](./docs/step3-config.png)

#### 推广计划配置
- **计划名称** — 自定义名称，建议格式：`SKU_日期_版本`
  - 例：`Summer_Collection_20260401`
- **每日预算** — 每天的投放预算（USD）
  - 建议：$50-500/天
  - 可随时在后台调整

#### 受众定向

**年龄范围**
- 设置目标受众的年龄段
- 通常 25-45 岁转化率最高

**性别**
- 可选：男性、女性、或全部
- 根据产品特性选择

**国家/地区**
- 输入国家代码，用逗号分隔
- 例：`US, CA, UK, AU`
- 支持多个地区

**兴趣标签**
- 输入相关兴趣，用逗号分隔
- 例：`shopping, fashion, beauty`
- 3-5 个最佳

#### 转化配置

**落地页URL**（最重要！）
- 输入转化页面的完整URL
- 建议添加 UTM 参数追踪来源
- 例：`https://shop.com/product?utm_source=tiktok&utm_medium=ads`

**广告文案**（可选）
- 短文本描述（50-200字）
- 例：`Limited time! 30% off summer collection`

**行动号召按钮**
- SHOP_NOW — 立即购物（电商首选）
- LEARN_MORE — 了解更多
- SIGN_UP — 立即注册
- DOWNLOAD — 下载应用
- 等等

---

### 步骤4：预览和提交

![Step 4](./docs/step4-preview.png)

**预览内容：**
- ✓ 创意信息（文件名、大小）
- ✓ 计划配置（名称、预算）
- ✓ 受众定向（年龄、性别、地区等）
- ✓ 转化配置（落地页、CTA等）

**确认无误后：**
1. 点击"🚀 创建广告"按钮
2. 系统会自动：
   - 上传创意文件到TikTok
   - 创建推广计划（CONVERSION目标）
   - 创建广告组（配置受众定向）
   - 创建广告（绑定创意和落地页）

**提交后：**
- 页面显示 ✅ 成功
- 获得 Campaign ID、Ad Group ID、Ad ID
- 可以在 TikTok 后台或本系统的仪表板查看

## 成功后的下一步

### 1. 查看仪表板
点击"📊 查看仪表板"，看你的广告：
- 实时数据监控
- ROAS、花费、转化等关键指标
- 自动优化建议

### 2. 继续创建
点击"➕ 创建新广告"，为同一商品创建多个变体：
- 不同地区的受众
- 不同创意素材
- A/B 测试

### 3. 监控和优化
系统会自动：
- 同步数据（每30分钟）
- 检测异常（CPM上升、Hook Rate低等）
- 给出 LLM 优化建议

## 常见问题

### Q1: 广告创建后多久能看到数据？
**A:** 24-48 小时后才能积累足够数据。建议先运行 3-7 天观察效果。

### Q2: 创意上传失败怎么办？
**A:** 
- 检查文件大小（视频<2GB，图片<30MB）
- 检查文件格式（MP4等）
- 检查网络连接
- 查看浏览器控制台的错误信息

### Q3: 如何修改已创建的广告？
**A:** 在 `/dashboard` 仪表板中：
- 查看广告详情
- 点击调整预算、暂停等操作
- LLM 会给出优化建议

### Q4: 可以上传同一个创意多次吗？
**A:** 可以。上传过的创意会获得 video_id，创建新广告时直接使用。

### Q5: 支持多账户吗？
**A:** 支持。每次创建广告时选择不同的账户即可。

## 表单验证

### 步骤1：选择账户
- ❌ 必须选择一个账户
- ❌ 账户状态必须是"授权有效"

### 步骤2：上传创意
- ❌ 必须上传视频或图片
- ❌ 文件大小必须符合限制
- ✅ 可选：只上传一个创意

### 步骤3：广告配置
- ❌ 计划名称必填
- ❌ 每日预算必填（>0）
- ❌ 落地页URL必填且有效
- ✅ 受众定向有默认值
- ✅ 广告文案可选

### 步骤4：确认
- ✓ 所有信息都已预览
- ✓ 点击"创建广告"提交

## 技术细节

### Vue 3 特性
- `<script setup>` — 现代 API
- `ref()` / `reactive()` — 状态管理
- 事件处理 — @click, @change 等
- 条件渲染 — v-if, v-else-if, v-else
- 循环 — v-for
- 绑定 — v-model

### 集成的 API
```javascript
advertiserApi.list()        // 获取账户列表
adsApi.uploadVideo(...)     // 上传视频
adsApi.uploadImage(...)     // 上传图片
adsApi.createFullCampaign() // 创建完整广告
```
